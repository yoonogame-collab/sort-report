氣泡排序法 (Bubble Sort)

氣泡排序法是一種簡單的排序演算法。它會反覆比較相鄰的元素，若順序錯誤則交換，直到整個序列有序。

演算法步驟

  1. 從陣列的第一個元素開始，依序比較相鄰的元素。
  2. 如果前一個元素比後一個元素大，則交換它們的位置。
  3. 重複上述步驟，直到陣列的最後一個元素。
  4. 對剩下的元素重複此過程，直到整個陣列有序。
  5. 如果在某一回合中沒有發生交換，表示陣列已經有序，可以提早結束。

用 C 語言實現的程式碼

#include <stdio.h>

// 氣泡排序函式
void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int swapped = 0; // 標誌是否有交換
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = 1;
            }
        }
        if (!swapped) break; // 如果沒有交換，提早結束
    }
}

// 主函式
int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);

    printf("原始陣列: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    bubbleSort(arr, n);

    printf("排序後陣列: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}

執行結果

假設輸入的陣列為 {64, 34, 25, 12, 22, 11, 90}，執行後的輸出結果如下：

原始陣列: 64 34 25 12 22 11 90
排序後陣列: 11 12 22 25 34 64 90

優缺點

  • 優點：實現簡單，適合用於教學或小型資料集。
  • 缺點：效率較低，時間複雜度為 O(n²)，不適合大型資料集。
