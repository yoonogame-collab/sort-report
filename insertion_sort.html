📌 插入排序法 (Insertion Sort)

觀看演算法步驟的交互式演示

插入排序法是一種簡單直觀的排序演算法。它通過逐一取未排序元素，並將其插入已排序的部分中的正確位置，最終完成排序。

演算法步驟

  1. 從第二個元素開始（第一個元素視為已排序）。
  2. 選取當前元素作為待插入元素。
  3. 從已排序部分的末尾開始比較，找到合適的插入位置。
  4. 將待插入元素插入到該位置，已排序部分的元素後移。
  5. 重複步驟 2-4，直到所有元素都被處理。

互動演示

用 C 語言實現的程式碼

#include <stdio.h>

// 插入排序函式
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        
        // 將大於 key 的元素向右移動
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        // 將 key 插入到正確位置
        arr[j + 1] = key;
    }
}

// 主函式
int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr) / sizeof(arr[0]);

    printf("原始陣列: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    insertionSort(arr, n);

    printf("排序後陣列: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}

優缺點

  • 優點：實現簡單，對小型資料集或近似有序的資料集效率較高；原地排序，不需額外空間。
  • 缺點：時間複雜度為 O(n²)，不適合大型資料集。
  • 應用場景：適合用於教學或對小型資料集排序，以及在實時數據流中進行增量排序。
