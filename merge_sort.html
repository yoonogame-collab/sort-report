🧩 合併排序法 (Merge Sort)

觀看演算法步驟的交互式演示

合併排序法是採用分治法 (Divide and Conquer) 的典型應用。將大問題分成小問題解決，再將結果合併。

演算法步驟

  1. 將陣列從中間切分成左右兩個子陣列。
  2. 遞迴地對左右子陣列進行合併排序。
  3. 將兩個已排序的子陣列合併成一個完整的排序陣列。

複雜度分析

最佳時間複雜度 O(n log n)
平均時間複雜度 O(n log n)
最差時間複雜度 O(n log n)
空間複雜度 O(n)
穩定性 穩定 (Stable)

用 C 語言實現的程式碼

void merge(int arr[], int l, int m, int r) {
    int n1 = m - l + 1;
    int n2 = r - m;
    int L[n1], R[n2];
    for (int i = 0; i < n1; i++) L[i] = arr[l + i];
    for (int j = 0; j < n2; j++) R[j] = arr[m + 1 + j];

    int i = 0, j = 0, k = l;
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) arr[k++] = L[i++];
        else arr[k++] = R[j++];
    }
    while (i < n1) arr[k++] = L[i++];
    while (j < n2) arr[k++] = R[j++];
}

void mergeSort(int arr[], int l, int r) {
    if (l < r) {
        int m = l + (r - l) / 2;
        mergeSort(arr, l, m);
        mergeSort(arr, m + 1, r);
        merge(arr, l, m, r);
    }
}

優缺點

  • 優點：時間複雜度穩定為 O(n log n)；屬於穩定排序。
  • 缺點：需要額外的 O(n) 空間來儲存暫存陣列。
