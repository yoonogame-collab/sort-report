🎯 選擇排序法 (Selection Sort)

觀看演算法步驟的交互式演示

選擇排序法是每一輪從未排序部分找到最小值，和當前位置交換，直到排序完成。

演算法步驟

  1. 從第 0 個位置開始，找出未排序區間 (i 到末尾) 的最小值。
  2. 若最小值位置不是 i，則與 i 位置交換。
  3. i 右移一位，重複直到最後一位。

複雜度分析

最佳時間複雜度 O(n²)
平均時間複雜度 O(n²)
最差時間複雜度 O(n²)
空間複雜度 O(1)
穩定性 不穩定 (Unstable)

用 C 語言實現的程式碼

#include <stdio.h>

// 選擇排序函式
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIdx]) {
                minIdx = j;
            }
        }
        // 將找到的最小值與目前位置交換
        int temp = arr[minIdx];
        arr[minIdx] = arr[i];
        arr[i] = temp;
    }
}

int main() {
    int arr[] = {64, 25, 12, 22, 11};
    int n = sizeof(arr) / sizeof(arr[0]);

    selectionSort(arr, n);

    printf("排序後的陣列: \n");
    for (int i = 0; i < n; i++)
        printf("%d ", arr[i]);
    return 0;
}

優缺點

  • 優點：邏輯簡單，原地排序（In-place），不需要額外空間；在數據交換次數上表現較優。
  • 缺點：不論資料初始狀態如何，時間複雜度均為 O(n²)，效能較差；屬於不穩定排序。
